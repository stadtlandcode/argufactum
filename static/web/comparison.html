Argufactum - Vergleichstabelle Prototyp
Dieser Prototyp befindet sich noch in einer frühen Entwicklungsphase. Aktuell können Kriterien und Alternativen hinzugefügt, bearbeitet und entfernt werden sowie grundlegende Datentypen ausgewählt werden. Daneben ist die Gewichtung und Auswertung der Daten funktionsfähig. Die Daten werden im LocalStorage gespeichert.


Die nächsten geplanten Features sind:

  • Speicherung der Vergleichstabelle unter einer eindeutigen ID auf einem Server, um die Vergleichstabelle teilen zu können
  • Integration weiterer grundlegender Datentypen wie Zahlen und Freitext
  • Auswahl der eigenen Präferenz je Kriterium ("was ist gewünscht?")
